Comme nous l'avons dit dans notre dossier, nous avons utilisés différents outils pour mener à bien notre projet.
Nous avons souvent changés d'avis, mais pour preuve que tout ceci ne repossait pas uniquement sur de simples allégations,
nous avons rassemblé tout notre code au sein d'un même projet (réalisé sous IntelliJ pour information).
La classe JDOM_TAL correspond à notre tout premier travail avec JDOM.
Les classes des packages Element_XML et TAL correspondent à l'utilisation de SAX.
Et pour finir, la classe Test_NLP correspond à notre travail sur les dépendances, directement avec StanfordNLP.
